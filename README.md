# Vim Config without Plugin

Although I am using neovim, I think it is completely fine for you to put all the code in .vimrc

# Code Runner

* Python, C/C++

  You can press `[r` to run python, C/C++ code.

* Latex, Typst

  Latex and Typst will auto compile when you save it with `:w`
